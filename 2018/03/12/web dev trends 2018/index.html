Web Dev Trends 2018

Ada beberapa trend pada web developer yang terjadi pada tahun 2018 ini,yaitu,

The Basics

HTML,CSS,JS itu adalah basic dari web dev,meskipun banyak teknologi,framework,dll basic tetap jadi yang terpenting, karena bagaimanapun sebelum bisa jadi ahli di teknologi yang baru,developer tetap harus tahu bagaimana HTML, CS, dan JS bisa bekerja.

Node.js

Node.js sudah populer pada beberapa tahun terakhir ini pada bahasa untuk back-end, meskipun tidak mengganggu kepopuleran PHP, Ruby dll terutama dimana Serverless Web sedang naik daun seperti saat ini, hal ini karena tidak ada kelebihan yang cukup signifikan dibanding framework diatas, dan alasan kenapa Node.js tetap populer adalah karena javascript menjadi semakin penting akhir akhir ini.

React, Angular dan Vue.js

Ketika Node.js peruntukan nya untuk back-end, untuk kedua framework dan satu library (react) ini untuk front-end , alasannya sama, karena javascript semakin penting dan tentu saja karena kemampuannya untuk menampilkan dynamic elements dengan semakin mudah dan powerful.

Web Components & Component Compiler

Kedua hal tersebut menjadi semakin populer karena pertama, Web Components yang dibuat dari vanilla JS dan bisa digunakan diluar framework, kedua, karena 4 wep API sudah disupport oleh browser besar seperti firefox, chrome dll, ketiga, karena kepopuleran dari Web Components, Components Compiler menjadi mengikutinya karena kemudahan yang ditawarkan pada compiling Web Components .

Static Page Generator

Semakin banyak pages maka semakin banyak JS yang dirender, dan semakin rendah pula performa, dengan static page generator kita bisa merender web kita di server dan kita bisa deploy static file saja, efisiensi bertambah tanpa mengurangi kualitas, bukan karena itu saja, namun juga karena hal ini semakin memudahkan Serverless Apps yang akan kita bahas nantinya.

Progressive Web Apps

Idenya adalah menggabungkan mobile app dan web, karena ternyata pengguna web jauh lebih banyak daripada pengguna mobile app yang harus diinstall, karena lebih mudah tentu saja, menjadikan PWAs menjadi populer akhir akhir ini.

Serverless Apps

Serverless bukan berarti tanpa server sepenuhnya, tapi ini tentang tanpa management server karena server akan berubah tergantung isinya, dan kita hanya menggunakannya saat ada user yang mengakses nya, tentu saja biayanya pun tergantung user nya juga, gabungan dari kemudahan dan biaya rendah menjadikannya po puler.

Artificial Intelligence dan Bots

Ini tentu saja karena kemudahan dan fitur yang ditawarkan, misal chat bot yang bisa membantu user di web kita, atau kita kenal Machine Learning, Big Data, dll ini menjadikannya primadona di 2018, sebelum 2018 bahkan kemungkinan besar sesudahnya karena ini adalah masa depan.

Kesimpulan

Dari kesemua itu tentu saja kita tidak harus ahli hanya untuk mengikuti trend, tapi jika kita bisa belajar beberapa dari mereka tentu saja akan sangat baik.
